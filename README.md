# 🎵 音乐播放器

一个纯前端的音乐播放器，支持自动从 GitHub 仓库加载音乐文件，部署在 GitHub Pages 上使用。

## ✨ 功能特性

- 🎧 **自动加载音乐** - 从 GitHub 仓库的 music 目录自动加载所有音频文件
- 🎵 **多种格式支持** - 支持 MP3、FLAC、WAV 等格式
- AA **支持歌词** - lrc格式的歌词(与原曲同名, 同目录)
- 🔄 **播放控制** - 播放/暂停、上一首/下一首、进度条拖动、快进快退
- 🔀 **播放模式** - 支持顺序播放、随机播放、列表循环、单曲循环
- 🔊 **音量控制** - 音量滑块、静音切换、键盘快捷键
- 📋 **播放列表** - 显示所有歌曲，点击即可播放
- 💾 **本地存储** - 自动保存用户设置（音量、播放模式等）
- ⌨️ **键盘快捷键** - 完整的键盘操作支持
- 📱 **响应式设计** - 适配移动端和桌面端

## 🚀 快速开始

### 1. 克隆或下载项目

```bash
git clone https://github.com/patiencewu/music.git
cd music
```

### 2. 添加音乐文件

将你的音乐文件放入 `music/` 目录：

```
music/
├── index.html
├── styles.css
├── player.js
├── config.json
└── music/
    ├── 歌曲1.flac
    ├── 歌曲2.mp3
    ├── 歌曲2.lrc
    ├── song
        └──歌曲3.mp3
    ├── abc
        └──歌曲3.flac
    ├── ...
    └── ...
        └──...
```

### 3. 配置仓库信息（可选）

编辑 `config.json` 文件：

```json
{
  "repository": {
    "owner": "patiencewu",
    "repo": "music",
    "branch": "main",
    "path": "music"
  }
}
```

**注意：** 如果部署到 GitHub Pages，播放器会自动检测仓库信息，无需手动配置。

### 4. 部署到 GitHub Pages

1. 将项目推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择要发布的分支（通常是 main）
4. 访问 `https://patiencewu.github.io/music/`

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `空格` | 播放/暂停 |
| `←` | 上一首 |
| `→` | 下一首 |
| `↑` | 音量增加 |
| `↓` | 音量减少 |
| `M` | 静音/取消静音 |
| `S` | 随机播放开关 |
| `R` | 循环模式切换 |

### 加载优先级

播放器会按以下顺序尝试加载播放列表：

1. **GitHub API** - 自动从仓库的 music 目录加载所有音频文件
2. **playlist.json** - 从本地配置文件加载
3. **默认列表** - 使用内置的默认歌曲

## 🎯 使用方法

### 添加新音乐

只需将音乐文件放入 `music/` 目录，然后点击播放列表的刷新按钮即可。

### 刷新播放列表

点击播放列表右上角的刷新按钮，播放器会重新从 GitHub 仓库加载音乐文件。

### 播放控制

- 点击播放按钮或按空格键播放/暂停
- 点击进度条跳转到指定位置
- 使用音量滑块调节音量
- 点击随机/循环按钮切换播放模式

## 📝 注意事项

1. **公开仓库**：确保你的仓库是公开的，否则无法通过 GitHub API 访问
2. **API 限制**：GitHub API 有速率限制，匿名请求每小时最多 60 次
3. **文件大小**：GitHub Pages 对单个文件大小有限制（建议小于 100MB）
4. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）

## 🐛 故障排除

### 无法加载音乐文件
- 检查仓库是否为公开
- 检查 config.json 配置是否正确
- 查看浏览器控制台的错误信息

### 播放列表为空
- 确认 music 目录下有音频文件
- 检查文件扩展名是否在支持的格式列表中
- 尝试刷新播放列表

### GitHub API 请求失败
- 检查网络连接
- 确认仓库信息正确
- 可能是 API 速率限制，稍后再试
